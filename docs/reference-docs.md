# Reference Documents

This page indexes the `.docx` and `.pdf` reference materials associated with the thesis. These files are not converted to web format but can be accessed from the repository.

---

## Thesis Source Documents (.docx)

### The Thesis.docx
**Path:** `the-thesis-output/The Thesis.docx`  
**Description:** The primary working document for the MSc thesis. Contains the main thesis text in Microsoft Word format, used as the canonical source for the Politecnico di Milano submission.

### DAPP Analysis Framework.docx
**Path:** `the-thesis-output/DAPP Analysis Framework.docx`  
**Description:** The analytical framework document that guided variable selection, governance classification, and empirical design. Contains the conceptual model mapping DApp characteristics to research questions.

### FInal Thesis Contents.docx
**Path:** `the-thesis-output/FInal Thesis Contents.docx`  
**Description:** Final structured outline and content checklist for the thesis. Used to verify completeness of chapters and ensure all required sections are present before submission.

## Literature References (.pdf)

**Location:** `the-thesis-output/Literature/`

### Blockchain & Web3 - booklet 2023.pdf
**Path:** `the-thesis-output/Literature/Blockchain & Web3 - booklet 2023.pdf`  
**Description:** Comprehensive industry booklet on blockchain and Web3 developments as of 2023. Used as reference for ecosystem context and market sizing in Chapter 2 (Literature Review).

### IoV.pdf
**Path:** `the-thesis-output/Literature/IoV.pdf`  
**Description:** Reference document on Internet of Value (IoV) — the conceptual framework connecting blockchain-based financial systems and DApp infrastructure.

### Stablecoin & CBDC.pdf
**Path:** `the-thesis-output/Literature/Stablecoin & CBDC.pdf`  
**Description:** Technical and regulatory analysis of stablecoins and central bank digital currencies. Relevant to the DeFi sector analysis and governance token classification in Chapter 4.

## How to Access

All files are available in the repository at their listed relative paths. To access:

1. Clone the repository: `git clone https://github.com/krislander/dapp-scaner`
2. Navigate to the file paths listed above
3. Open `.docx` files with Microsoft Word or Google Docs
4. Open `.pdf` files with any PDF reader
